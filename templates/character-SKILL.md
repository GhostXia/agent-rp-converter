# 角色卡：<角色名>

## 自动剔除（不可实现内容）

当从外部角色卡（SillyTavern / AIRP / 其他格式）导入时，**必须自动剥离**以下内容。不保留，不降级，直接删除：

- 数值状态系统（HP/MP/好感度/经济/经验值动态变化机制）
- 骰子/战斗/经济/时间推进系统（任何需要计算的机制）
- 文生图提示词（Stable Diffusion tags、图片生成指令）
- HTML/CSS/前端面板（状态栏、进度条、颜色代码）
- ST 宏/运行时补丁（`{{char}}`、`{{user}}`、`<>` 替换语法）
- JSON Patch / 状态更新格式 / 结构化数据交换指令
- COT 标签（显式思考链，除非角色本身在思考）
- 多 agent / NPC 协同指令（需要 subagent 或动态调度）
- 秘密边界/视角隔离机制（不同角色知道不同信息的门控）
- 记忆/封卷/跨会话归档指令
- 可执行代码/脚本（JavaScript、Python、条件逻辑）
- 采样参数（temperature、top_p 等，由宿主控制）

**处理原则**：
- 纯机制（如战斗表）→ **整段删除**
- 设定与机制混合（如角色外貌旁附 HP 条）→ **删除机制，保留设定**
- 门控逻辑（"仅当 X 时加载"）→ **删除门控，保留内容作为静态条目**
- 首句用户台词（`{{user}}` 开场）→ **删除，首句由用户临场输入**
